# Bug Bounty Curriculum (Year-long)

## Month 1-2: Web Fundamentals

### 1. HTTP/HTTPS
- Basics: Methods, Status Codes, Headers, Cookies
- HTTP/2 & HTTP/3
- Tools: Burp Suite, Wireshark
- [MDN Web Docs](https://developer.mozilla.org/en-US/docs/Web/HTTP)

### 2. Web Browsers
- Same-origin policy
- Cross-Origin Resource Sharing (CORS)
- Security features: CSP, HSTS, WebSockets
- [Browser Internals](https://browser.engineering/)

### 3. Web Architecture
- Web servers: Apache, Nginx, etc.
- Content Delivery Network (CDN) & Reverse Proxies
- Serverless architectures & Cloud platforms
- [High Performance Browser Networking](https://hpbn.co/)

### 4. Server-side
- Frameworks: Express, Django, Flask, Rails
- RESTful APIs & GraphQL
- MVC structure

### 5. Client-side
- Deep dive into HTML, CSS, JS
- Modern JS frameworks: React, Angular, Vue
- DOM manipulation & event handling

## Month 3-4: Common Web Vulnerabilities

### 1. Authentication and Session Management Bugs
- Password reset flaws
- Session fixation
- JWT issues

### 2. Access Control Bugs
- Privilege escalation
- IDORS
- Client-side access control bugs
- Paywall bypass
- [Portswigger Academy](https://portswigger.net/web-security)

### 3. Data Exposure
- Sensitive information in URLs or error messages
- Misconfigured cloud storage

### 4. Vulnerable Components
- Using components with known vulnerabilities

### 5. Hacking & Practice
- Platforms: [Hack The Box](https://www.hackthebox.eu/), [TryHackMe](https://tryhackme.com/), [PentesterLab](https://pentesterlab.com/)
